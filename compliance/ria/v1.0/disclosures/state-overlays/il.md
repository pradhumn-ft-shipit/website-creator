# State overlay — Illinois (IL)

**Applies when:** advisor is **state-registered** (AUM < $100M) with a primary state of Illinois (§5.5).
State-registered investment advisers in Illinois register with the **Illinois Securities Department (Office of
the Illinois Secretary of State)**.

## Footer registration line (replaces the generic state line)

> {{firm_name}} is registered as an investment adviser with the Illinois Securities Department. Registration
> does not imply a certain level of skill or training.

## State-specific notes (verify with counsel before publish)
