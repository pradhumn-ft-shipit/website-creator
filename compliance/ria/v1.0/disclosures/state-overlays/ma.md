# State overlay — Massachusetts (MA)

**Applies when:** advisor is **state-registered** (AUM < $100M) with a primary state of Massachusetts (§5.5).
State-registered investment advisers in Massachusetts register with the **Massachusetts Securities Division
(Office of the Secretary of the Commonwealth)**.

## Footer registration line (replaces the generic state line)

> {{firm_name}} is registered as an investment adviser with the Massachusetts Securities Division. Registration
> does not imply a certain level of skill or training.

## State-specific notes (verify with counsel before publish)

- Confirm current Massachusetts Securities Division disclosure rules under the Massachusetts Uniform Securities
  Act (M.G.L. c. 110A / 950 CMR) before publish.
- No state-specific *website* prohibitions beyond the federal set are encoded in v1.0; flag if review identifies any.
