# State overlay — New Jersey (NJ)

**Applies when:** advisor is **state-registered** (AUM < $100M) with a primary state of New Jersey (§5.5).
State-registered investment advisers in New Jersey register with the **New Jersey Bureau of Securities (Division
of Consumer Affairs)**.

## Footer registration line (replaces the generic state line)

> {{firm_name}} is registered as an investment adviser with the New Jersey Bureau of Securities. Registration
> does not imply a certain level of skill or training.
