# State overlay — Florida (FL)

**Applies when:** advisor is **state-registered** (AUM < $100M) with a primary state of Florida (§5.5).
State-registered investment advisers in Florida register with the **Florida Office of Financial Regulation (OFR)**.

## Footer registration line (replaces the generic state line)

> {{firm_name}} is registered as an investment adviser with the Florida Office of Financial Regulation.
> Registration does not imply a certain level of skill or training.

## State-specific notes (verify with counsel before publish)

- Confirm current Florida OFR disclosure requirements under Chapter 517, Florida Statutes, before publish.
- No state-specific *website* prohibitions beyond the federal set are encoded in v1.0; flag if review identifies any.
